# A Collection of AI coding agent tools

This repo contains plugins and skills that I use some of the time.

## Using the plugins with Claude Code

Add this repository as a Claude Code plugin marketplace, then install the Fusion Advisor plugin. Claude Code reads the root `.claude-plugin/marketplace.json`, which points to the plugin under `plugins/fusion-advisor/`.

```sh
/plugin marketplace add YaserJaradeh/ai-tools
/plugin install fusion-advisor@ai-tools
```
